# 减脂 + 逆转脂肪肝计划

制定日期：2026-04-20

## 基本情况

| 指标 | 数值 |
|------|------|
| 身高 / 体重 | 168cm / 70kg |
| BMI | 24.8（亚洲标准已超重，正常 <23.9） |
| 目标体重 | 62-63kg（BMI ≈ 22） |
| 需减掉 | 7-8kg |
| 健康问题 | 轻度脂肪肝 |

**关键认知**：体重减掉 5-7%（约 3.5-5kg）肝脏脂肪就会显著下降；减掉 7-10% 轻度脂肪肝基本可逆。

---

## 饮食原则

### 热量目标
**1600-1800 kcal/天**，制造约 400 kcal 缺口，每周减 0.3-0.5kg。

### 早餐（必须改）
换掉 Bagel，选以下其一：
- 燕麦片 + 2 个煮鸡蛋 ≈ 290 kcal ✓
- 无糖希腊酸奶（150g）+ 蓝莓 ≈ 200 kcal
- 全麦吐司 1 片 + 鸡蛋 + 蔬菜 ≈ 350 kcal

牛角包、Bagel、面包点心**不能当正餐**。

### 午餐
- 米饭限 1 碗（150g 熟饭），能吃杂粮饭更好
- 减少粉丝/面条，用蔬菜替代
- 汤底优先清汤

### 晚餐（关键控制点）
- **20:00 后不吃**
- 蛋白质 + 蔬菜为主，少主食或不吃主食
- 目标 500-600 kcal 以内

### 要少吃 / 不吃
| 食物 | 原因 |
|------|------|
| 牛角包、Bagel、年轮蛋糕 | 高脂+精制糖，直接入肝合成脂肪 |
| 粉丝、白米线 | 高 GI，升甘油三酯 |
| 麻辣烫（碳水为主版） | 高热量+高钠 |
| 薯条 | 高脂精制碳水 |
| 含糖饮料、果汁 | 果糖是脂肪肝最直接推手 |

### 要多吃（对肝脏友好）
- 深海鱼（三文鱼、金枪鱼）— omega-3 抗肝脏炎症
- 西兰花、菠菜、白菜 — 肝脏解毒
- 鸡蛋（不限黄）
- 绿茶 — 保护肝细胞
- 坚果少量（核桃、杏仁，10-15 颗/天）
- 美式咖啡 — 对肝脏有保护作用 ✓

| 类型 | 频率 | 时长 |
|------|------|------|
| 力量训练 | 3x/周 | 60 min |
| 有氧（跑步/快走） | 2-3x/周 | 30-40 min |

每周总运动消耗目标：**800-1000 kcal**

## 时间线

| 阶段 | 周数 | 目标体重 | 里程碑 |
|------|------|---------|--------|
| 快速启动 | 1-8 周 | ～66-67kg | 减 3-4kg，肝脏脂肪开始明显下降 |
| 稳步推进 | 9-16 周 | ～64-65kg | 接近目标 |
| 巩固达标 | 17-24 周 | 62-63kg | 复查肝脏 B 超 |

**建议第 8-12 周做一次肝脏 B 超复查。**
